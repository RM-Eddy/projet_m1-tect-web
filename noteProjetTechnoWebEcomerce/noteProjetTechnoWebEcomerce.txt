* Créer le Backend (NestJS) :
** Installer le CLI NestJS :
npm i -g @nestjs/cli

** Créer le projet backend: 
nest new backend

** Se placer dans le dossier
cd backend

* Installer les dépendances PostgreSQL et Swagger : 
* PostgreSQL + TypeORM
npm install @nestjs/typeorm typeorm pg

* Swagger
npm install --save @nestjs/swagger swagger-ui-express

* Exécute cette commande dans le dossier backend :
npm install class-validator class-transformer


Dépendance	Rôle principal				Utilisation typique

class-validator	Valider les champs d’un DTO			@IsNotEmpty(), @IsEmail(), etc.
class-transformer	Transformer un objet brut en instance de classe	plainToInstance() ou via ValidationPipe
